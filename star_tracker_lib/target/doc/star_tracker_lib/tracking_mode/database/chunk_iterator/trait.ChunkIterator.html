fn find_close_ref_region( &self, find: Radians, tolerance: Radians, found: &mut dyn List<SearchResult> )

Finds close matches to the provided angular separation and returns the star pair reference. The an element of the star pair reference can be inserted into find_star to get the actual location.

Arguments
  • find - The angular separation between the found stars to find in the database.
  • found - The closest matches to the provided find.
Returns

The fields at the end of each pair. Use this information so that when finding multiple pairs connected, they can span multiple regions.
